Automated System for a Drycleaning

I'm part of a team of four people working on this ongoing project. A system that consists of an internal software application and a website for a business portfolio is being developed for a dry cleaners. For this project, we are collaborating with a real-world client. We are completing this project by using UML patterns.

Four phases of this project are Inception, Elaboration, Construction and Transition.

Inception Phase: During this stage, we spoke with our client and compiled a rough feature list of everything they requested. After that, all of the requirements were gathered, including with use cases and additional specifications, and inception documentation was completed. To reduce any future issues that might occur during development, a Risk List and RequirementsÂ ranking list were made as parts of this document.

Elaboration Phase: For the main risk use cases, we created classes, implementation diagrams, database entities, and stored procedures during this step. To make sure that the fundamental functionality is operating, test procedures were designed and testing was conducted.

Construction phase is still ongoing.

Tools used:

Visual Studio, DBMS, ASP.NET Razor Pages, HTML, CSS, C#, Javascript, T-SQL, etc.

Additional Notes:

This project is currently in progress and we are working on the development of the software application.
